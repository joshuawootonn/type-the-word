## January Updates

### The Retypeability Update

TTW's old progression UX worked for first-time completion but did not support intentional retyping well.

Changes shipped:

- Added a chapter-level "Log" so repeated typing sessions are recorded.
- Adjusted verse dimming so it now reflects in-progress chapter state instead of permanent historical dimming.
- Updated `/history` book progression to use resettable box-based progress instead of capped percentages.
